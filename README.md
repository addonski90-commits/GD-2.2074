# How to install GD 2.2074

## Step 1: Choose a version
First choose which version you want (**Unmodded** or **Modded**).

[Unmodded](https://drive.google.com/uc?export=download&id=191YTi-paKqGD0jOVP1YbKwo39E8y2Ahb)
[Modded](https://drive.google.com/uc?export=download&id=1b2hgOsqEDjKuYC7CZi4zfFBcwsMBGHSz)

---

## Step 2: Open the ZIP file
Open your **Downloads** folder and select the ZIP file.

<img src="https://i.imgur.com/ckjtBIl.png" alt="ZIP file in downloads" width="50%">

---

## Step 3: Extract the files
Right-click the ZIP file and choose **Extract All**.

<img src="https://i.imgur.com/1hNR1vM.png" alt="Extract all" width="50%">

Press **Extract**.

<img src="https://i.imgur.com/2bh6FF6.png" alt="Press extract">

---

## Step 4: Start the game
Open `GeometryDash.exe`.

<img src="https://i.imgur.com/Bi6MaxN.png" alt="Start Geometry Dash" width="50%">

---

## Step 5 (Modded only): Add mods
If you chose the **modded version**, press the **Geode** button to add mods.

<img src="https://i.imgur.com/yRRgnJl.png" alt="Geode mod menu" width="50%">
